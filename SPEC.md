# AVProc - 音视频处理桌面应用规格说明

## 1. 项目概述

- **项目名称**: AVProc
- **项目类型**: 桌面应用程序 (Go + Fyne)
- **核心功能**: 音视频格式预览、转换、剪裁、拼接
- **目标用户**: 需要简单音视频处理功能的普通用户

## 2. 技术栈

- **GUI框架**: Fyne (跨平台Go GUI库)
- **音视频处理**: FFmpeg (通过命令行调用)
- **构建工具**: Go 1.21+

## 3. UI/UX 规格

### 3.1 窗口结构

- **主窗口**: 900x650 像素，可调整大小
- **布局**: 标签页式界面 (Tabbed layout)

### 3.2 页面结构

```
┌─────────────────────────────────────────────┐
│  菜单栏 (文件 | 帮助)                         │
├─────────────────────────────────────────────┤
│  [预览] [转换] [剪裁] [拼接]                   │
├─────────────────────────────────────────────┤
│                                             │
│              内容区域                         │
│                                             │
└─────────────────────────────────────────────┤
│  状态栏                                       │
└─────────────────────────────────────────────┘
```

### 3.3 页面详情

#### 3.3.1 预览页面 (Preview)
- 文件选择按钮
- 显示文件信息:
  - 文件名、格式、时长
  - 分辨率、帧率
  - 编码器、比特率
  - 音频轨道信息
- 视频缩略图/播放预览

#### 3.3.2 转换页面 (Convert)
- 源文件选择
- 输出格式选择: MP4, AVI, MKV, MP3, WAV, AAC
- 输出质量: 高/中/低
- 输出路径选择
- 转换按钮 + 进度条

#### 3.3.3 剪裁页面 (Crop)
- 视频文件选择
- 时间范围输入:
  - 开始时间 (时:分:秒)
  - 结束时间 (时:分:秒)
- 输出文件名
- 剪裁按钮 + 进度条

#### 3.3.4 拼接页面 (Merge)
- 文件列表 (支持拖拽添加)
- 上移/下移按钮
- 移除选中按钮
- 输出文件名
- 拼接按钮 + 进度条

### 3.4 视觉设计

## 4. 功能规格

### 4.1 格式预览
- 使用 ffprobe 获取媒体信息
- 解析并显示:
  - 视频: 编解码器、分辨率、帧率、比特率、时长
  - 音频: 采样率、声道数、编码器、时长

### 4.2 格式转换
- 支持格式: MP4, AVI, MKV, MP3, WAV, AAC
- 使用 ffmpeg 进行转码
- 支持质量选择 (CRF值调整)

### 4.3 视频剪裁
- 基于时间范围的剪裁
- 使用 -ss 和 -t 参数
- 保持原始编码格式

### 4.4 视频拼接
- 支持多个文件拼接
- 需要相同格式
- 使用 ffmpeg concat demuxer

## 5. 错误处理

- FFmpeg 不存在: 提示用户安装
- 文件路径错误: 显示错误提示
- 转换失败: 显示详细错误信息
- 不支持的格式: 明确提示

## 6. 依赖要求

- FFmpeg (必须): 系统命令 `ffmpeg` 和 `ffprobe`
- Go >= 1.21
- fyne.io/fyne v2

## 7. Windows 支持

- 使用 LLVM MinGW 进行 CGO 编译
- 内置播放器实现，无需 mpv
- 打包脚本自动包含 FFmpeg 二进制文件
- 文件对话框支持过滤器筛选
